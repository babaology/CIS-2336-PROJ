# What This Project Does

This is a web application for the UH Sugar Land Student Center. It allows students to learn about the center, browse available rooms, 
and submit room booking requests online

# Features

- Sticky navigation bar with scroll 
- Mobile-responsive hamburger menu
- FAQ — only one item open at a time
- Room booking form with full client-side validation
- Remotely hosted images and embedded YouTube video
- Form data sent to Express backend 
- Developer Contact section on homepage
